# Workflow

MCP Gateway Registry Desk is a hosted remote MCP for AI SDK MCP gateway registry.

## Repeatable Flow

1. Open MCP Gateway Registry Desk and select the buyer plan.
2. Create or request a bearer token from the hosted product.
3. Add https://mcpgatewaydesk.clauxel.com/mcp to a compatible MCP client.
4. Run tools/list, then call list_team_tools with public-safe sample data.
5. Save the returned receipt or export for human review.

## Output Mindset

The useful artifact is not a marketing claim. It is evidence that a reviewer can inspect, export, and compare later.
